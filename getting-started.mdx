## Quick Setup

Get started with Alexandra Janney Documentation in just a few minutes. Follow these steps to create your first project and begin documenting.

<Steps>
  <Step title="Create Account" icon="user-plus">
    Visit the dashboard and sign up for a new account using your email address.
  </Step>
  <Step title="Verify Email" icon="mail">
    Check your email for a verification link and confirm your account.
  </Step>
  <Step title="Set Up Profile" icon="settings">
    Complete your profile with your name and preferences.
  </Step>
  <Step title="Create First Project" icon="folder-plus">
    Click "New Project" and give it a descriptive name.
  </Step>
</Steps>

## Configuration Options

Customize your documentation space to fit your workflow. Here are the key configuration settings:

<Tabs>
  <Tab title="Basic Settings" icon="settings">
    Configure project name, description, and basic metadata.
    
    <CodeGroup tabs="JSON,YAML">
      ```json
      {
        "name": "My Project",
        "description": "Documentation for my API",
        "version": "1.0.0"
      }
      ```
      ```yaml
      name: My Project
      description: Documentation for my API
      version: 1.0.0
      ```
    </CodeGroup>
  </Tab>
  <Tab title="Advanced Options" icon="sliders">
    Set up webhooks, integrations, and automation rules.
    
    Use the advanced settings to connect with external tools.
  </Tab>
</Tabs>

<Callout kind="info">All configuration changes take effect immediately across your documentation space.</Callout>

## Troubleshooting Common Issues

If you encounter problems during setup, check these common solutions:

| Issue | Solution |
|-------|----------|
| Email not received | Check your spam folder and verify the email address |
| Project creation fails | Ensure you have the necessary permissions |
| Configuration not saving | Try refreshing the page and re-entering the settings |
